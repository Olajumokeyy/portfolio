Portfolio - Next.js + TypeScript + Tailwind (Waterfalls pastel)
==============================================================

Quick start:

1. Install dependencies
   npm install
2. Run locally
   npm run dev
3. Deploy to Vercel
   - Push to GitHub and import the repo in Vercel

Edit your profile in:
  src/data/profile.ts
Add assets to:
  public/profile.jpg
  public/resume.pdf
